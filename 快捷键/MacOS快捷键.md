# Mac快捷键

||功能|快捷键|
|--|--|--|
|⭐️|弹出 emoji 面板|`Ctrl + CMD + Space`|
||退出应用全屏|`Control + Command + F`|


## 截图
||功能|快捷键|
|--|--|--|
||截取整个屏幕|`⌘ Command + ⇧ Shift + 3`|
|⭐️|选择区域截图|`Command + ⇧ Shift + 4`|
||窗口截图|`⌘ Command + ⇧ Shift + 4 + Space`|
||截图进剪贴板|`⌘ Command + ⇧ Shift + 4 + Ctrl`|
|⭐️|截图工具面板|`⌘ Command + ⇧ Shift + 5`|
