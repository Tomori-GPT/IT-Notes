# jar包 / war包

- JAR = Java 程序包（Java Archive）
- WAR = Web 应用包（Web Application Archive）

它们本质上都是ZIP压缩包，只是用途不同。

### jar包
运行方式：
java -jar myapp.jar
```
Java程序
    │
    ├── 打包成 JAR
    │
    └── 双击/Java命令运行

包括：
myapp.jar
├── META-INF
│     └── MANIFEST.MF
│
├── com/
│     └── example/
│            Main.class
│            User.class
│
└── lib（有时候）
```

### war包
运行方式：
java -jar myapp.jar
```
Java Web项目
    │
    ├── 打包成 WAR
    │
    └── 部署到 Tomcat

包括：
myapp.jar
├── META-INF
│     └── MANIFEST.MF
│
├── com/
│     └── example/
│            Main.class
│            User.class
│
└── lib（有时候）
```

### 区别对比:
|JAR|WAR|
|--|--|
|Java程序|Java Web应用|
|给 JVM 运行|给 Tomcat 等 Servlet 容器部署
|可以 java -jar|一般部署到服务器运行
|内容主要是 class|class + HTML + JSP + CSS + JS 等
|一般有 Main 方法|一般没有 Main 方法（传统项目）
